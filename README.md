# somniverse-app.github.io

Site officiel de l'application [Somniverse](https://somniverse-app.github.io), généré avec Jekyll et déployé sur GitHub Pages.

## Lancer le projet localement

### Prérequis

- [Ruby](https://www.ruby-lang.org/fr/documentation/installation/)
- [Bundler](https://bundler.io/) (`gem install bundler`)

### Installation et démarrage

```bash
# Installer les dépendances
bundle install

# Lancer le serveur local
bundle exec jekyll serve
```

Le site est accessible sur **http://localhost:4000**.

> Le serveur surveille les modifications et reconstruit automatiquement le site. Un redémarrage est nécessaire après toute modification de `_config.yml`.
